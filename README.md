# Potfolio Website


A professional **Personal Portfolio Website** built using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

##  Project Overview

This portfolio website is designed to showcase personal information, skills, projects, resume, and contact details in a modern and responsive layout. It features a premium user interface, smooth scrolling, theme switching, mobile navigation, and scroll reveal animations.

##  Features

* Responsive personal portfolio layout
* Hero section with introduction and call-to-action buttons
* About section
* Skills section
* Projects showcase section
* Contact section
* Resume download button
* Sticky navigation bar
* Dark / Light theme toggle
* Mobile menu
* Smooth scrolling
* Scroll reveal animations
* Clean modern glassmorphism-inspired design

##  Technologies Used

* **HTML5** – page structure
* **CSS3** – layout, styling, responsiveness, animations
* **JavaScript** – mobile menu, theme toggle, reveal animations

##  How to Run the Project

1. Download or clone this repository.
2. Open the folder in **VS Code**.
3. Add your images in the `images` folder and place your CV as `resume.pdf`.
4. Open `index.html` using **Live Server**.

##  Sections Included

* **Home / Hero Section** – short introduction and CTA buttons
* **About Section** – personal introduction
* **Skills Section** – technical skills and tools
* **Projects Section** – portfolio projects with buttons for live demo and GitHub
* **Contact Section** – email, LinkedIn, GitHub, and location
* **Footer** – copyright section

Before publishing the portfolio:

* Replace placeholder contact details with your own
